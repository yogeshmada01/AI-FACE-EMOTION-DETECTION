# AI Face Emotion Detection 🎭

A real-time face emotion detection system using DeepFace, OpenCV, and TensorFlow.

## 🚀 Features
- Real-time face detection via webcam
- Emotion classification (Happy, Sad, Angry, Surprise, Neutral)
- Live visualization with confidence scores

## 🛠 Tech Stack
- Python
- OpenCV
- DeepFace
- TensorFlow

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/yogeshmada01/AI-FACE-EMOTION-DETECTION

2. Install dependencies:
   pip install deepface opencv-python

3. Run the project:
   python main.py
